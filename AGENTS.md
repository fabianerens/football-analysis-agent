## Context

This is an project that demonstrates how to build a sequential agent pipeline using Google's Agent Development Kit (ADK) with Gradio web interface. 

Follow the instructions in https://raw.githubusercontent.com/google/adk-docs/main/docs/agents/workflow-agents/sequential-agents.md



## Project Management

The project is managed using `uv`. 

Use uv add to add new dependencies and uv update to update existing ones.
